# screenshots/

Still images captured on the **Ubuntu 24.04 + LxQt target VM** (4 vCPU / 8 GB),
as required by the Challenge 02 submission checklist and About JioPC §4.1.

Capture these from a real run (`jiopc-agent --config ... --analyse`) and drop the
PNGs here:

1. `01-run-streaming.png` — the live `[PASS]/[FAIL]/[BLOCKED]` lines streaming
   during Part A / Part B / Part C.
2. `02-part-b-apps.png` — native app windows launched during Part B (Firefox,
   VLC, LibreOffice, Files, etc.) visible on the desktop.
3. `03-summary.png` — the final summary line (`NN/43 passed … exit_code=N`).
4. `04-llm-verdict.png` — the LLM analysis output ending in the bold
   **PROMOTE** / **HOLD** recommendation.

> These are captured on the VM and are intentionally not generated by the repo
> code. The performance numbers in `benchmarks/REPORT.md` were measured on the
> same VM.
